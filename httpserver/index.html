# httpServer使用技巧

前提 全局安装http-server https://www.npmjs.com/package/http-server (opens new window)

**执行示例：**http-server ./ -P https://www.demo.com (opens new window)

使用步骤：

进入到需要启动的目录，执行http-server ./

如果需要代理，比如本地找不到的资源需要代理到线上域名 使用 -P https://www.demo.com (opens new window)
